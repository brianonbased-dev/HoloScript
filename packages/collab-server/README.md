# @holoscript/collab-server

Lightweight WebSocket relay server for HoloScript Studio multi-user collaboration.

## Usage

```bash
pnpm install
pnpm start
```

The server relays WebSocket messages between connected HoloScript Studio clients for real-time collaborative editing.

## Development

```bash
pnpm dev        # Start with file watching
pnpm build      # Bundle with esbuild
```
