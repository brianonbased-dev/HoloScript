# @holoscript/studio-desktop

HoloScript Studio native desktop application built with Tauri 2. Wraps the Studio web frontend in a native window with file system, shell, and dialog access.

## Usage

```bash
pnpm install
pnpm build        # Build production binary
```

Requires Rust toolchain and Tauri CLI prerequisites.

## Development

```bash
pnpm dev          # Launch dev mode with hot reload
pnpm tauri        # Run Tauri CLI commands directly
```
