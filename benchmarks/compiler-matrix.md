## How to Update Snapshots

When making an intentional compiler output change, regenerate with:

```bash
pnpm --filter @holoscript/core exec vitest run -u \
  src/compiler/__tests__/golden-output/golden.test.ts
```

Commit both the updated `golden.test.ts.snap` and the code change together so reviewers can see the diff.

## Promoting Targets to Golden Tests

To add a new target to the golden regression suite:
1. Add a test case to `packages/core/src/compiler/__tests__/golden-output/golden.test.ts`
2. Run `vitest run -u` to generate its snapshot
3. Update the table above
