# R3F Renderer

**Shared React Three Fiber renderer components for HoloScript.**

## Overview

`@holoscript/r3f-renderer` provides renderer-facing components and integration helpers for React Three Fiber based applications.

## Installation

```bash
npm install @holoscript/r3f-renderer
```

## Use When

- Your app is built on React Three Fiber.
- You need renderer components shared across HoloScript web surfaces.
- You want a React-native way to host HoloScript scenes.

## Key Capabilities

- Shared R3F rendering integration.
- Useful for previews, studio-adjacent web tools, and app shells.
- Bridges HoloScript scene data into established React 3D workflows.
