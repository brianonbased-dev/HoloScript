# Marketplace Web

**Web UI for browsing, searching, and installing marketplace traits and packages.**

## Overview

`@holoscript/marketplace-web` is the frontend experience for marketplace interactions, sitting on top of the marketplace and registry service layers.

## Installation

```bash
npm install @holoscript/marketplace-web
```

## Use When

- You need a browser UI for marketplace workflows.
- You want a reference frontend for package discovery.
- You are building a hosted or self-hosted marketplace experience.

## Key Capabilities

- Search and browse interfaces for marketplace content.
- Install and discover flows for community packages.
- Frontend companion to backend registry and marketplace APIs.
