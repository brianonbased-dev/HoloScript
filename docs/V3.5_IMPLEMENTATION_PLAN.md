### Agent 1: LOD Streaming System
**Duration**: ~35 minutes
**Files**: New files in `packages/core/src/lod/`
**Lines**: ~1,200 total

**Tasks**:
1. **LODStreamingManager.ts** (~400 lines)
   - Async mesh loading from disk/network
   - Priority queue based on distance + screen coverage
   - Resource budget management (memory + bandwidth)
   - Prefetching prediction based on camera velocity

   - GPU-driven frustum culling

3. **LODCache.ts** (~250 lines)
   - LRU cache for LOD meshes
   - Compressed format support (Draco, meshopt)
   - Texture streaming coordination
   - Memory budget enforcement

4. **Tests** (~200 lines, 25 tests)
   - Streaming priority queue correctness
   - GPU culling integration tests
   - Cache eviction behavior
   - Memory leak prevention

**Integration Points**:
- LODManager (existing) - Triggers streaming requests
- RenderNetworkTrait - Render job coordination
- AdvancedCompression (v3.3) - Draco/KTX2 support

**Expected Performance**:
- 50+ high-poly dragons at 60 FPS (vs 47 before)
- 70% reduction in CPU LOD selection time (GPU-driven)
- 50% reduction in memory usage (streaming + cache)
