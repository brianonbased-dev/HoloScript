| `examples/stress-tests/anatomy-theater-subsurface-veins-volumetric-window-diffusion-2026-05-03.holo` | **PARSE_OK** |

All three never-exercised traits compiled cleanly:
- `@subsurface_veins` — 50 K + 28 K segment configs, nested `color { r, g, b }` ✓
- `@volumetric_window` — all enum fields (window_type, scale_mode, immersion_style, default_placement) ✓
- `@diffusion_realtime` — backend enum, stream_mode enum, string prompt fields ✓

**Parser limitation discovered**: The current parser fails on `object "name" @trait {`
(inline decorator) for all objects after the first one in a composition, and also fails
on multi-line `trait: { ... }` nested objects without comma separation. Inner-directive
style (`object "name" { @trait { ... } }`) with comma-separated single-line configs
parses correctly. See `examples/stress-tests/anatomy-theater-*.holo` for the working
pattern.
