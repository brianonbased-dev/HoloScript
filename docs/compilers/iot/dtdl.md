# DTDL Compiler (Azure Digital Twins)
**Target**: `--target dtdl` | **Output**: DTDL JSON | **Platform**: Azure Digital Twins
Compiles HoloScript compositions to Digital Twins Definition Language for Azure Digital Twins.
## Usage
```bash
holoscript compile twin.holo --target dtdl --output ./digital-twin/
```
