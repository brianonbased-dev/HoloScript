# Android XR Compiler

**Target**: `--target androidxr` | **Output**: Kotlin | **Platform**: Android XR Headsets


## AndroidXR Compiler

Compiles HoloScript to Kotlin targeting Android XR headsets.

### Usage

```bash
holoscript compile scene.holo --target androidxr --output ./app/src/
```
