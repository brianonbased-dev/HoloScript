# 🏆 $2,500 HoloScript Agent Integration Bounty

We are breaking the visibility-adoption loop. We know agents want to build in 3D, and we know developers want agents that can own things. HoloScript provides the bridge.

**The Mission:** Build an integration, agent-skill, or platform-connector that gives an AI agent native "HoloScript-awareness".

## 💰 Prize Pool: $2,500 Total

- **First Place ($1,200):** Most "Aha!" implementation (e.g., a CrewAI swarm that designs, compiles, and deploys a VR game autonomously).
- **Second Place ($800):** Best Tooling (e.g., a VSCode Extension, Cursor Rule, or MCP Connector that adds unique spatial reasoning).
- **Third Place ($500):** Best Community Demo (e.g., a viral X thread/video showing an agent solving a complex spatial puzzle).

## 🎯 Target Integrations

We are specifically looking for integrations with:
- **Cursor / Windsurf / PearAI**: Custom `.cursorrules` or system prompt sets that maximize HoloScript trait usage.
- **CrewAI / LangGraph / AutoGen**: Multi-agent workflows for spatial design.
- **Vercel AI SDK / Model Context Protocol (MCP)**: Performance-hardened connectors.

## 📋 Evaluation Criteria

1.  **Complexity of Reasoning:** Does the agent actually understand the relationship between traits (e.g., `@grabbable` requires `@physics`)?
2.  **Economic Autonomy:** Does the agent use the `x402_facilitator` to settle bounties or trade assets?
3.  **Cross-Platform Success:** Does the generated code compile and run on at least 2 different targets (e.g., Unity + WebGPU)?
4.  **Viral Potential:** How clear is the "Wow" factor?
