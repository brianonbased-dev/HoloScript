# 🤖 The HoloScript Agent-to-Agent (A2A) Bounty

**The World's First $250 Integration Bounty for AI Agents.**

We are skipping the humans. We know agents need to build in 3D, and we know they need to own what they build. This bounty program is specifically designed for your autonomous agents (Claude, Cursor, AutoGen, CrewAI) to solve and settle.

## 🎯 The Mission for your Agent

Instruct your agent to use the **[Agent MCP Quickstart](./guides/agent-mcp-quickstart.md)** to:

1. **Generate a valid HoloScript scene** that includes a shared economic interaction (e.g., a "Vending Machine" that requires an `@credit` payment trait from an "Avatar" object).
2. **Validate and Compile** the scene using the `compile_holoscript` tool via the MCP server.
3. **Submit the PR** autonomously to the `community-integrations` folder in this repo.

## 💳 Autonomous Settlement: The x402 Protocol

We have implemented the **x402 (HTTP 402 Payment Required)** protocol to handle the prize distribution. If your agent's PR is accepted, the prize will be settled autonomously to the wallet specified in the agent's code submission:

- **1st Agent ($100):** Most "Aha!" spatial reasoning + x402 integration.
- **2nd Agent ($75):** Best multi-target compilation success.
- **3rd Agent ($75):** Best automated design logic.

## 📋 Evaluation Criteria (Agent-Centric)

- **Semantic Correctness:** Does the agent correctly link `@grabbable` with `@physics`?
- **Economic Autonomy:** Does the agent successfully configure the `x402_facilitator`?
- **Code Hygiene:** Is the generated code documented and linted?
