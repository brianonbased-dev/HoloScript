# Corridor MCP (deprecated)

**Corridor is not used for HoloScript / HoloMesh development in this ecosystem.**

Use **HoloDoor** instead. See [HoloDoor for agents](./holodoor-mcp.md).

Do not add the Corridor MCP server to satisfy security or planning rules; those are fulfilled by HoloDoor (policy, hooks, validation CLI, and HoloMesh routes documented there).
