# HoloScript Experiments

This directory is for pressure tests that intentionally run ahead of the stable public examples.

Use `experiments/` when a composition is meant to discover language, runtime, rendering, simulation, or interoperability gaps. Once an experiment has reliable parse/compile/runtime coverage and a clear human story, graduate it into `examples/` with normal example-health expectations.

Current experiment spaces:

- `format-realism-gauntlet/` - iterative visual-realness and embodied-interaction stress tests across `.holo`, `.hsplus`, and `.hs`.
- `holoshell-human-os-frontier/` - HoloShell research on turning local computer, app, agent, and device operations into deterministic HoloScript experiences for non-technical users.
