# HoloShell Idea Seeds

## Agent Hardware Orchestra

A non-technical user asks for an outcome. HoloShell decomposes the job across
local agents and surfaces it as one deterministic HoloScript experience.

- Codex: hardware validation, builds, local execution, GPU/WASM reality.
- Claude: long-context architecture and cross-file reasoning.
- VS Code/Copilot: inline edits and developer cockpit.
- Shell agents: OS actions, installs, process control, filesystem operations.
- Browser agents: visual/browser verification.
- HoloLand agents: world embodiment, spatial surfaces, player/creator UX.

The human sees roles, steps, approvals, artifacts, warnings, receipts, and replay.

## File World

Turn folders, downloads, project assets, photos, and documents into a navigable
HoloScript world. Files become objects with meaning, provenance, relationships,
age, duplication state, sensitivity, and safe actions.

## Slow Computer Clinic

Wrap process, memory, GPU, disk, network, startup, and package-manager diagnostics
into a room where the user can see bottlenecks and choose safe remedies.

## Build Machine Room

A HoloScript/HoloLand project setup becomes a visible machine: dependency check,
install, build, parse, compile, render, screenshot, task filing, and receipt.

## Cloud Skeleton Wrapper

Expose fragmented Google, Microsoft, Apple, browser, account, permission, and
file-format workflows as deterministic HoloScript plans. AI handles provider
quirks behind the scenes; the user sees stable controls and receipts.

## Device Control With Safety Envelopes

Wrap local devices, robots, printers, cameras, headsets, and sensors as HoloScript
actors with permissions, intent, preview, execution, receipt, and rollback notes.
