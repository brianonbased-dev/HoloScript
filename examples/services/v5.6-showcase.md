# v5.6 "Observable Platform" — Showcase

**Time to First Hello World: ~5 minutes**

## What's New

v5.6 adds production-grade observability to the HoloScript agent orchestration stack:

- **OTLP/HTTP Export** — Ship trace spans to Jaeger, Grafana Tempo, or any OTLP collector
- **W3C Trace Context** — Distributed tracing across agent delegations via `traceparent`/`tracestate`
- **Prometheus Metrics** — Counters, gauges, histograms with text exposition format
- **Structured Logging** — JSON log entries with trace correlation (traceId, spanId)
- **Health Endpoints** — `/health` with subsystem checks, `/metrics` with Prometheus text
- **4 MCP Tools** — `query_traces`, `export_traces_otlp`, `get_agent_health`, `get_metrics_prometheus`

## Quick Start

### Step 1: Register agents and run a workflow

```typescript
import {
  getDefaultRegistry,
  getTelemetryCollector,
  getPrometheusMetrics,
  TraceContextPropagator,
  StructuredLogger,
} from '@holoscript/core';

// Register an agent
const registry = getDefaultRegistry();
registry.register({
  id: 'sensor-01',
  name: 'SensorAgent',
  version: '1.0.0',
  capabilities: [{ type: 'analyze', domain: 'iot', name: 'collect' }],
  endpoints: [{ protocol: 'local', address: 'local://sensor-01', primary: true }],
  trustLevel: 'local',
  status: 'online',
  tags: ['sensor', 'iot'],
});
```

### Step 2: Start a traced operation

```typescript
const collector = getTelemetryCollector();
const span = collector.startSpan('sensor-read', {
  agentId: 'sensor-01',
  kind: 'server',
});

// Do work...
collector.endSpan(span.id, 'ok');
```

### Step 3: Query traces via MCP

```json
{
  "tool": "query_traces",
  "arguments": { "agentId": "sensor-01", "limit": 10 }
}
```

### Step 4: Check Prometheus metrics

```json
{
  "tool": "get_metrics_prometheus",
  "arguments": {}
}
```

Returns Prometheus exposition text:
```
# HELP holoscript_delegation_total Total task delegations
# TYPE holoscript_delegation_total counter
holoscript_delegation_total{status="completed"} 5
holoscript_delegation_duration_ms_bucket{le="100"} 3
```

### Step 5: Export to OTLP

```json
{
  "tool": "export_traces_otlp",
  "arguments": {
    "endpoint": "http://localhost:4318/v1/traces",
    "serviceName": "my-iot-pipeline"
  }
}
```

## Architecture

```
┌─────────────┐     ┌──────────────────┐     ┌────────────────┐
│ Agent Code  │────▸│ TelemetryCollect │────▸│  OTLPExporter  │──▸ Jaeger/Tempo
│             │     │  spans + events  │     │  batch + retry │
└─────────────┘     └──────────────────┘     └────────────────┘
                           │                        │
                    ┌──────┴──────┐          ┌──────┴──────┐
                    │ Prometheus  │          │ Structured  │
                    │  Metrics    │          │   Logger    │
                    │ counters/   │          │  JSON +     │
                    │ histograms  │          │  traceId    │
                    └─────────────┘          └─────────────┘
                           │
                    ┌──────┴──────┐
                    │  /metrics   │──▸ Prometheus/Grafana
                    │  /health    │──▸ Load balancer
                    └─────────────┘
```
