# v6.0 "Universal Semantic Platform" -- Showcase

**Time to First Hello World: ~5 minutes**

## What's New

v6.0 is the graduation release unifying 9 phases of incremental evolution:

| Phase | Codename | Key Additions |
|-------|----------|---------------|
| v5.1 | Stabilization | AGENTS.md, SKILL.md bridge, A2A protocol |
| v5.2 | Parser Universalized | 239 traits, DialectRegistry, NodeServiceCompiler |
| v5.3 | Tooling as Semantic Bridge | Changesets, MCP quality gates, 3 MCP tools |
| v5.4 | Domains Unified | 31 domain types, 73 constraints, 72 LSP completions |
| v5.5 | Agents as Universal Orchestrators | FederatedRegistry, TaskDelegation, SkillWorkflow |
| v5.6 | Observable Platform | OTLP, Prometheus, W3C Trace Context, structured logging |

**Total: 122 MCP tools, 419+ new tests, 69 new files across 5 major subsystems.**

## Quick Start

### Step 1: Start the dev server

```bash
holoscript serve --watch --open
```

This starts the DevServer with SSE-based HMR. Edit any `.holo`/`.hs`/`.hsplus` file and see changes reflected instantly.

### Step 2: Explore the API

```json
{
  "tool": "get_api_reference",
  "arguments": { "format": "markdown" }
}
```

Returns comprehensive API documentation for all 122 MCP tools, organized by category.

### Step 3: Run an agent workflow

```json
{
  "tool": "compose_workflow",
  "arguments": {
    "steps": [
      { "tool": "discover_agents", "input": { "capability": "analyze" } },
      { "tool": "delegate_task", "input": { "task": "sensor-read" } }
    ]
  }
}
```

### Step 4: Check observability

```json
{
  "tool": "query_traces",
  "arguments": { "limit": 10 }
}
```

### Step 5: Inspect the trace waterfall

```json
{
  "tool": "inspect_trace_waterfall",
  "arguments": {
    "spans": [
      { "traceId": "t1", "spanId": "s1", "name": "orchestrator", "kind": "server", "startTime": 0, "endTime": 500, "status": "ok" }
    ]
  }
}
```
