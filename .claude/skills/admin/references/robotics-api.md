## 🚀 Quick Start

```python
from holoscript.robotics import export_urdf, export_sdf, generate_ros2_launch

# 1. Export to URDF
holo_code = """
sphere {
    @physics
    @collidable
    @position(0, 1, 0)
}
"""

urdf_result = export_urdf(holo_code, robot_name="my_robot")
if urdf_result.success:
    print(urdf_result.urdf_content)
    print(f"Links: {urdf_result.link_count}, Joints: {urdf_result.joint_count}")

# 2. Export to SDF (Gazebo)
sdf_result = export_sdf(holo_code, world_name="my_world")
if sdf_result.success:
    print(sdf_result.sdf_content)

# 3. Generate ROS 2 Launch File
launch_result = generate_ros2_launch(
    holo_code,
    package_name="my_robot_description",
    use_rviz=True,
    use_gazebo=True
)
print(launch_result.launch_content)
```

## 📚 API Reference

### Core Functions

#### `export_urdf()`

Export HoloScript code to URDF (Unified Robot Description Format).

**Signature:**
```python
def export_urdf(
    holo_code: str,
    robot_name: str = "holoscript_robot",
    include_visual: bool = True,
    include_collision: bool = True,
    include_inertial: bool = True,
    default_mass: float = 1.0,
    mesh_path_prefix: str = "package://robot_description/meshes",
) -> URDFExportResult
```

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `holo_code` | `str` | *required* | HoloScript source code to export |
| `robot_name` | `str` | `"holoscript_robot"` | Name for the URDF `<robot>` element |
| `include_visual` | `bool` | `True` | Include `<visual>` geometry elements |
| `include_collision` | `bool` | `True` | Include `<collision>` geometry elements |
| `include_inertial` | `bool` | `True` | Include `<inertial>` (mass/inertia) properties |
| `default_mass` | `float` | `1.0` | Default mass in kg for objects without `@physics` |
| `mesh_path_prefix` | `str` | `"package://robot_description/meshes"` | ROS package path prefix for mesh files |

**Returns:** `URDFExportResult`

**Example:**
```python
result = export_urdf(
    holo_code="""
        cube { @physics @collidable @position(0, 1, 0) }
    """,
    robot_name="test_robot",
    include_visual=True,
    include_collision=True,
    include_inertial=True,
    default_mass=2.5
)

if result.success:
    with open("robot.urdf", "w") as f:
        f.write(result.urdf_content)
    print(f"✅ Exported {result.link_count} links, {result.joint_count} joints")
else:
    print(f"❌ Errors: {result.errors}")
```

**URDF Output Structure:**
```xml
<?xml version="1.0"?>
<!-- Auto-generated by HoloScript Python Bindings -->
<robot name="holoscript_robot">
  <material name="default">
    <color rgba="0.8 0.8 0.8 1.0"/>
  </material>

  <link name="base_link"></link>

  <link name="cube">
    <visual>
      <geometry><box size="1 1 1"/></geometry>
      <material name="default"/>
    </visual>
    <collision>
      <geometry><box size="1 1 1"/></geometry>
    </collision>
    <inertial>
      <mass value="1.0"/>
      <inertia ixx="0.1" ixy="0.0" ixz="0.0" iyy="0.1" iyz="0.0" izz="0.1"/>
    </inertial>
  </link>

  <joint name="base_to_cube_joint" type="fixed">
    <parent link="base_link"/>
    <child link="cube"/>
    <origin xyz="0 0 0" rpy="0 0 0"/>
  </joint>
</robot>
```

#### `export_sdf()`

Export HoloScript code to SDF (Simulation Description Format) for Gazebo.

**Signature:**
```python
def export_sdf(
    holo_code: str,
    world_name: str = "holoscript_world",
    sdf_version: str = "1.9",
    include_physics: bool = True,
    include_gravity: bool = True,
) -> SDFExportResult
```

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `holo_code` | `str` | *required* | HoloScript source code to export |
| `world_name` | `str` | `"holoscript_world"` | Name for the SDF `<world>` element |
| `sdf_version` | `str` | `"1.9"` | SDF format version (1.9 for Gazebo Harmonic) |
| `include_physics` | `bool` | `True` | Include physics engine configuration |
| `include_gravity` | `bool` | `True` | Include gravity vector (`<gravity>`) |

**Returns:** `SDFExportResult`

**Example:**
```python
result = export_sdf(
    holo_code="""
        plane { @collidable @static }
        sphere {
            @physics
            @gravity
            @collidable
            @position(0, 5, 0)
        }
    """,
    world_name="physics_demo",
    sdf_version="1.9",
    include_physics=True,
    include_gravity=True
)

if result.success:
    with open("world.sdf", "w") as f:
        f.write(result.sdf_content)
    print(f"✅ Exported {result.model_count} models")
```

**SDF Output Structure:**
```xml
<?xml version="1.0"?>
<!-- Auto-generated by HoloScript Python Bindings -->
<sdf version="1.9">
  <world name="holoscript_world">
    <gravity>0 0 -9.8</gravity>

    <physics name="default_physics" type="ode">
      <max_step_size>0.001</max_step_size>
      <real_time_factor>1</real_time_factor>
      <real_time_update_rate>1000</real_time_update_rate>
    </physics>

    <include><uri>model://ground_plane</uri></include>
    <include><uri>model://sun</uri></include>

    <model name="sphere">
      <static>false</static>
      <pose>0 0 0 0 0 0</pose>
      <link name="link">
        <inertial>
          <mass>1.0</mass>
          <inertia>
            <ixx>0.1</ixx><ixy>0.0</ixy><ixz>0.0</ixz>
            <iyy>0.1</iyy><iyz>0.0</iyz><izz>0.1</izz>
          </inertia>
        </inertial>
        <visual name="visual">
          <geometry><sphere><size>1 1 1</size></sphere></geometry>
        </visual>
        <collision name="collision">
          <geometry><sphere><size>1 1 1</size></sphere></geometry>
        </collision>
      </link>
    </model>
  </world>
</sdf>
```

#### `generate_ros2_launch()`

Generate a ROS 2 Python launch file for a HoloScript scene.

**Signature:**
```python
def generate_ros2_launch(
    holo_code: str,
    package_name: str = "holoscript_scene",
    urdf_file: str = "robot.urdf",
    use_rviz: bool = True,
    use_gazebo: bool = True,
) -> ROSLaunchResult
```

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `holo_code` | `str` | *required* | HoloScript source code (used for metadata extraction) |
| `package_name` | `str` | `"holoscript_scene"` | ROS 2 package name containing generated files |
| `urdf_file` | `str` | `"robot.urdf"` | Filename of generated URDF (relative to `config/`) |
| `use_rviz` | `bool` | `True` | Include RViz2 visualization node |
| `use_gazebo` | `bool` | `True` | Include Gazebo Classic simulator |

**Returns:** `ROSLaunchResult`

**Example:**
```python
result = generate_ros2_launch(
    holo_code="cube { @physics }",
    package_name="my_robot_description",
    urdf_file="my_robot.urdf",
    use_rviz=True,
    use_gazebo=True
)

if result.success:
    with open("launch/holoscript.launch.py", "w") as f:
        f.write(result.launch_content)
    print(f"✅ Launch file created with nodes: {', '.join(result.nodes)}")
```

**Generated Launch File Structure:**
```python
"""
ROS 2 Launch File - Generated by HoloScript Python Bindings
Package: my_robot_description
"""

from launch import LaunchDescription
from launch.actions import DeclareLaunchArgument, IncludeLaunchDescription
from launch.substitutions import LaunchConfiguration, PathJoinSubstitution
from launch_ros.actions import Node
from launch_ros.substitutions import FindPackageShare
import os


def generate_launch_description():
    pkg_share = FindPackageShare(package="my_robot_description").find("my_robot_description")
    urdf_path = os.path.join(pkg_share, "config", "my_robot.urdf")

    with open(urdf_path, "r") as f:
        robot_description = f.read()

    nodes = []

    # Robot State Publisher
    nodes.append(Node(
        package="robot_state_publisher",
        executable="robot_state_publisher",
        name="robot_state_publisher",
        output="screen",
        parameters=[{"robot_description": robot_description}],
    ))

    # Joint State Publisher GUI
    nodes.append(Node(
        package="joint_state_publisher_gui",
        executable="joint_state_publisher_gui",
        name="joint_state_publisher_gui",
    ))

    # RViz2 Visualization
    nodes.append(Node(
        package="rviz2",
        executable="rviz2",
        name="rviz2",
        output="screen",
    ))

    # Gazebo Classic
    nodes.append(Node(
        package="gazebo_ros",
        executable="spawn_entity.py",
        arguments=["-topic", "robot_description", "-entity", "holoscript_robot"],
        output="screen",
    ))

    return LaunchDescription(nodes)
```

#### `list_robotics_traits()`

List HoloScript traits relevant to robotics simulation.

**Signature:**
```python
def list_robotics_traits() -> List[Dict[str, str]]
```

**Returns:** `List[Dict[str, str]]` with keys:
- `trait`: Trait name (e.g., `"@physics"`)
- `description`: Human-readable description
- `urdf_element`: Corresponding URDF element
- `sdf_element`: Corresponding SDF element

**Example:**
```python
traits = list_robotics_traits()
for t in traits:
    print(f"{t['trait']}: {t['description']}")
    print(f"  URDF: <{t['urdf_element']}>")
    print(f"  SDF:  <{t['sdf_element']}>")
```

**Output:**
```
@physics: Adds inertial properties to URDF/SDF link
  URDF: <inertial>
  SDF:  <inertial>
@collidable: Adds collision geometry to URDF/SDF link
  URDF: <collision>
  SDF:  <collision>
@static: Creates fixed joint in URDF; static model in SDF
  URDF: <fixed_joint>
  SDF:  <static=true>
@sensor: Adds sensor plugin hints
  URDF: <gazebo_sensor_plugin>
  SDF:  <sensor>
@grabbable: Interaction hint (comment-based in URDF/SDF)
  URDF: <!-- holoscript:grabbable -->
  SDF:  <!-- holoscript:grabbable -->
```

### Example 3: Complete ROS 2 Package Setup

```python
import os
from holoscript.robotics import export_urdf, generate_ros2_launch

# Mobile robot
mobile_robot = """
cube base {
    @physics
    @collidable
    @position(0, 0.2, 0)
    @scale(0.6, 0.2, 0.4)
}

cylinder wheel_fl {
    @physics
    @collidable
    @position(0.2, 0.1, 0.25)
    @rotation(90, 0, 0)
    @scale(0.1, 0.05, 0.1)
}

cylinder wheel_fr {
    @physics
    @collidable
    @position(0.2, 0.1, -0.25)
    @rotation(90, 0, 0)
    @scale(0.1, 0.05, 0.1)
}

cylinder wheel_rl {
    @physics
    @collidable
    @position(-0.2, 0.1, 0.25)
    @rotation(90, 0, 0)
    @scale(0.1, 0.05, 0.1)
}

cylinder wheel_rr {
    @physics
    @collidable
    @position(-0.2, 0.1, -0.25)
    @rotation(90, 0, 0)
    @scale(0.1, 0.05, 0.1)
}

cube sensor_mount {
    @sensor
    @position(0.3, 0.3, 0)
    @scale(0.1, 0.1, 0.1)
}
"""

# Create ROS 2 package structure
package_name = "mobile_robot_description"
os.makedirs(f"{package_name}/config", exist_ok=True)
os.makedirs(f"{package_name}/launch", exist_ok=True)

# Export URDF
urdf_result = export_urdf(
    mobile_robot,
    robot_name="mobile_robot",
    default_mass=2.0
)

with open(f"{package_name}/config/mobile_robot.urdf", "w") as f:
    f.write(urdf_result.urdf_content)

# Generate launch file
launch_result = generate_ros2_launch(
    mobile_robot,
    package_name=package_name,
    urdf_file="mobile_robot.urdf",
    use_rviz=True,
    use_gazebo=True
)

with open(f"{package_name}/launch/display.launch.py", "w") as f:
    f.write(launch_result.launch_content)

# Create package.xml
package_xml = f"""<?xml version="1.0"?>
<package format="3">
  <name>{package_name}</name>
  <version>1.0.0</version>
  <description>Generated by HoloScript Python Bindings</description>
  <maintainer email="dev@example.com">Developer</maintainer>
  <license>MIT</license>

  <buildtool_depend>ament_cmake</buildtool_depend>
  <exec_depend>robot_state_publisher</exec_depend>
  <exec_depend>joint_state_publisher_gui</exec_depend>
  <exec_depend>rviz2</exec_depend>
  <exec_depend>gazebo_ros</exec_depend>

  <export>
    <build_type>ament_cmake</build_type>
  </export>
</package>
"""

with open(f"{package_name}/package.xml", "w") as f:
    f.write(package_xml)

print(f"✅ ROS 2 package created: {package_name}/")
print(f"   - config/mobile_robot.urdf ({urdf_result.link_count} links)")
print(f"   - launch/display.launch.py ({len(launch_result.nodes)} nodes)")
print(f"   - package.xml")
print(f"\n🚀 Launch with: ros2 launch {package_name} display.launch.py")
```

## 🔧 Internal Helpers

The following internal functions are used by the public API:

| Function | Purpose |
|----------|---------|
| `_sanitize_xml_attr(value: str) -> str` | Sanitize strings for XML attribute values |
| `_sanitize_link_name(name: str) -> str` | Convert object names to valid URDF link names (lowercase, underscores) |
| `_get_geometry_for_object(obj_name: str) -> Dict[str, str]` | Map HoloScript object type to URDF/SDF geometry |

**Note**: Internal helpers are prefixed with `_` and not part of the public API.

## 🚨 Common Patterns & Gotchas

### Pattern: Check Success Before Using Results
```python
result = export_urdf(holo_code)
if result.success:
    # Safe to use result.urdf_content
    print(result.urdf_content)
else:
    # Handle errors
    print(f"Export failed: {result.errors}")
```

### Pattern: Custom Mass for Physics Objects
```python
# Use default_mass for all objects without explicit mass
result = export_urdf(
    holo_code,
    default_mass=5.0  # 5 kg for all objects
)
```

### Gotcha: URDF Requires base_link
URDF exports **always** include a `base_link` element for ROS compatibility. This is automatic and cannot be disabled.

### Gotcha: Parse Errors Return Failure Result
```python
# Invalid HoloScript code
result = export_urdf("invalid syntax here")
assert result.success == False
assert len(result.errors) > 0  # Contains parse errors
```

### Pattern: Minimal vs Full Export
```python
# Minimal URDF (no visual, collision, or inertial)
minimal = export_urdf(
    holo_code,
    include_visual=False,
    include_collision=False,
    include_inertial=False
)

# Full URDF (default - all elements included)
full = export_urdf(holo_code)
```
